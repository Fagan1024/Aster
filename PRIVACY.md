# Privacy

Aster is a local macOS previewer.

- Aster does not upload your files.
- Aster does not include analytics, tracking, or advertising.
- Imported files are copied to `~/Library/Application Support/Aster/Documents/`.
- The local file list is stored in `~/Library/Application Support/Aster/Library.json`.
- HTML files are rendered with WKWebView. If an HTML file contains remote resources, WebKit may request those resources just like a browser would.

---

## 中文

Aster 是一个本地 macOS 预览器。

- Aster 不会上传你的文件。
- Aster 不包含统计、追踪或广告。
- 导入的文件会复制到 `~/Library/Application Support/Aster/Documents/`。
- 本地文件列表保存在 `~/Library/Application Support/Aster/Library.json`。
- HTML 文件使用 WKWebView 渲染。如果 HTML 内包含远程资源，WebKit 可能会像浏览器一样请求这些资源。

## 日本語

Aster はローカルで動作する macOS プレビューアです。

- Aster はファイルをアップロードしません。
- Aster には分析、トラッキング、広告は含まれていません。
- 読み込んだファイルは `~/Library/Application Support/Aster/Documents/` にコピーされます。
- ローカルのファイル一覧は `~/Library/Application Support/Aster/Library.json` に保存されます。
- HTML ファイルは WKWebView で表示されます。HTML にリモートリソースが含まれている場合、WebKit はブラウザと同じようにそれらをリクエストすることがあります。
